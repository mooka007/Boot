Ninja Utility

A powerful command-line utility built with Node.js

Getting Started

Follow these steps to set up the Ninja Utility on your system:

1. Create Project Directory

$ mkdir ninja-utility
$ cd ninja-utility

2. Initialize Node.js Project

$ npm init -y

3. Install Dependencies

$ npm install axios chalk commander

Command Features

Greet Command

Displays a colorful greeting message in the terminal using Chalk.

$ node index.js greet
✨ Hello, Ninja Developer! ✨

greet.js

const chalk = require('chalk');

function greet() {
  console.log(
    chalk.bold(
      chalk.yellow('✨ Hello, Ninja Developer! ✨')
    )
  );
}

module.exports = greet;

Fetch Command

Fetches data from a public API using Axios and displays it.

$ node index.js fetch
Fetching data from API...
{ "data": [...] }

fetch.js

const axios = require('axios');

async function fetchData() {
  try {
    console.log('Fetching data from API...');
    const response = await axios.get('https://api.example.com/data');
    console.log(response.data);
  } catch (error) {
    console.error('Error fetching data:', error.message);
  }
}

module.exports = fetchData;

Read Command

Reads and displays the content of a specified file.

$ node index.js read example.txt
File content:
This is the content of the file.

read.js

const fs = require('fs');

function readFile(filePath) {
  try {
    const content = fs.readFileSync(filePath, 'utf8');
    console.log('File content:');
    console.log(content);
  } catch (error) {
    console.error('Error reading file:', error.message);
  }
}

module.exports = readFile;

Main Application

The index.js file ties everything together using the Commander package:

index.js

#!/usr/bin/env node
const { program } = require('commander');
const greet = require('./commands/greet');
const fetchData = require('./commands/fetch');
const readFile = require('./commands/read');

program
  .name('ninja-utility')
  .description('A CLI utility for various tasks')
  .version('1.0.0');

program
  .command('greet')
  .description('Display a greeting message')
  .action(greet);

program
  .command('fetch')
  .description('Fetch data from an API')
  .action(fetchData);

program
  .command('read <file>')
  .description('Read a file')
  .action((file) => readFile(file));

program.parse(process.argv);

Running the Utility

$ node index.js --help
Usage: ninja-utility [options] [command]

A CLI utility for various tasks

Options:
  -V, --version    output the version number
  -h, --help       display help for command

Commands:
  greet         Display a greeting message
  fetch         Fetch data from an API
  read <file>    Read a file
  help [command]  display help for command
